# Portfolio — dual-role (Support + Frontend / AI data)

React + Vite + TypeScript + Tailwind CSS v4 + Framer Motion.

## Customize

Edit **`src/content.ts`**: `soul` (human / first-person cards), `contact`, home coordinates for the distance sketch, pronunciation, desk photo path, and per-role copy (`support` vs `engineering`).

Header toggle switches the “lens” card and role-matched bullets; the rest of the page stays personal either way.

## Commands

```bash
npm install
npm run dev
npm run build
npm run preview
```
